## 📦 What this is

cyber-cultivation is a terminal RPG where you grow a cultivator with AI help. You make choices, build power, and move through a text-based world shaped by xianxia themes, tokens, and progression.

It runs in a terminal window and uses Claude Code as its core engine. The game is built for people who want a simple way to launch a fun AI game on their computer and play without a complex setup.

## 🧩 Main ideas in the game

### Cultivation path

Your character grows through stages of power. You start weak, then build skills, resources, and rank. Each step changes your stats and opens new choices.

### Token system

The game uses tokens as a game resource. You may spend them to advance, unlock actions, or shape your path through the world.

### AI guidance

Claude Code helps drive the experience. It can shape events, react to your choices, and keep the game moving with a dynamic feel.

### Terminal play

The full game runs in a terminal. This keeps the app simple and fast to launch on Windows.
